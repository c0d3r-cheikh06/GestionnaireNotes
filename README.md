# Gestionnaire de Notes

Application Android développée en Java dans le cadre de l'examen de TP Mobile 2025/2026.

Cette application permet de créer, modifier, supprimer et consulter des notes personnelles. Les données sont enregistrées localement avec SQLite afin d'être conservées même après la fermeture de l'application.

## Fonctionnalités

* Ajout de notes
* Modification de notes
* Suppression de notes
* Gestion des favoris
* Recherche par titre
* Choix de couleur
* Mode sombre
* Partage de notes
* Stockage SQLite

## Technologies utilisées

* Java
* Android Studio
* SQLite
* RecyclerView
